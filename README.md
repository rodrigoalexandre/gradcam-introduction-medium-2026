## 📝 **DESCRIPTION**

Este repositório contém o projeto criado para o hands-on apresentado no artigo **"Uma introdução à Explainable AI (XAI) utilizando Grad-CAM"** publicado no blog Medium em janeiro de 2026.

O artigo apresenta uma introdução à Explainable AI utilizando a técnica **Gradient-weighted Class Activation Mapping (Grad-CAM)** e uma demonstração do uso prático do Grad-CAM para explicar as predições de um modelo de Deep Learning.
<br><br>
## 📂 **PROJECT STRUCTURE**

|--- breastcancer-gradcam.ipynb (Jupyter notebook para treinamento do modelo e avaliação das predições)<br>
